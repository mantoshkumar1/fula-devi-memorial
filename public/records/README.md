# Public Records

This directory contains public records that document the work of Fula Devi
Memorial Sewa Sansthan.

Examples include:

- photographs;
- report cards;
- newspaper coverage;
- public guides;
- informational material.

These records support the public pages of the website and are preserved as part
of the institution's documented history.

Records are organised by programme, and by year where a programme runs on an
annual cycle.

Only publish material that:

- is accurate;
- respects privacy;
- has been approved for public release.

Publication is irreversible. Review every photograph and document for personal
data before it is added.
